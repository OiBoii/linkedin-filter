# Support (inRole)

## Get Help

When reporting issues, include:

- Extension version
- Browser version
- LinkedIn page URL pattern (e.g. jobs/search, jobs/collections)
- Steps to reproduce
- Screenshot or short screen recording

## Bug Reports

File a bug with:

- Expected behavior
- Actual behavior
- Any console errors from the page and extension service worker

## Feature Requests

Submit concise requests with user outcome and acceptance criteria.
