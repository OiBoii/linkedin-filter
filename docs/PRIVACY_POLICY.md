# Privacy Policy (inRole)

Effective date: 2026-02-23

inRole runs locally in your browser to filter LinkedIn job listing tiles.

## Data We Collect

inRole does not collect personal data and does not transmit data to external servers.

## Data Processed Locally

- Job listing tile text visible on LinkedIn pages (for filtering decisions).
- Extension settings you configure (keywords, excluded companies, toggles).

## Storage

- Settings are stored in `chrome.storage.sync` under your Chrome profile.
- No message content, credentials, profile data, or private LinkedIn messages are collected.

## Data Sharing

- No data is sold, shared, or transferred to third parties.

## Retention and Deletion

- Settings remain until changed, reset, or extension uninstall.
- Uninstall removes extension storage according to Chrome extension lifecycle behavior.

## Security

- No remote code execution patterns.
- No external script loading.
- Host access is limited to `https://www.linkedin.com/*`.

## Contact

For privacy questions, use support channel documented in `docs/SUPPORT.md`.
